# UW CSE 122 / CSE 123 Coursework

Java coursework from **CSE 122** and **CSE 123** at the University of Washington.

## Structure

### `cse122/` — Data Structures & Algorithms
| Folder | Topic |
|--------|-------|
| `C0_Media`, `C0_InvertedIndex`, `C0_SearchEngine` | Inheritance, interfaces, search indexing |
| `C2_Divide_Canvas`, `C2_Fill_Region`, `C2_Mondrian` | Recursion, graphics |

### `cse123/` — Programming Assignments
| Folder | Topic |
|--------|-------|
| `P0_Ciphers` | Caesar, substitution, and multi-cipher encryption |
| `P0_Debugging` | OOP debugging with class hierarchies |
| `P1_MiniGit` | Git-like version control system |
| `P2_DisasterRelief` | Graph pathfinding for disaster relief routing |
| `P3_SpamClassifier` | Decision tree email spam classifier |
| `Mini-Git` | Version of the mini-git implementation |

### `csc142/` — Intro Programming (CSC 142)
Foundational Java: file recursion, JavaFX templates.
